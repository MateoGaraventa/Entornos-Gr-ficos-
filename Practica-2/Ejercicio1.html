Ejercicio 1: Responder 1. ¿ Qué es CSS y para qué se usa? CSS, o Hojas de Estilo en Cascada (Cascading Style Sheets), es un lenguaje utilizado para describir el aspecto y formato de un documento HTML. Se usa para dar estilo a elementos HTML, como cambiar colores, fuentes, tamaños y diseños, para mejorar la apariencia visual de un sitio web.

2. CSS utiliza reglas para las declaraciones de estilo, ¿cómo funcionan? Las reglas CSS están compuestas por un selector y un bloque de declaraciones de estilo. El selector apunta a uno o varios elementos HTML, y el bloque de declaraciones especifica cómo se deben ver esos elementos. Cuando un navegador encuentra una regla CSS que coincide con un elemento en el HTML, aplica las declaraciones de estilo correspondientes a ese elemento.

3. ¿ Cuáles son las tres formas de dar estilo a un documento? Las tres formas de dar estilo a un documento son: Estilo en línea: Se aplican directamente a un elemento utilizando el atributo style en la etiqueta HTML. Estilo interno: Se define dentro del documento HTML, en la sección del encabezado . Estilo externo: Se define en un archivo CSS separado, que luego se enlaza con el documento HTML utilizando la etiqueta

4. ¿ Cuáles son los distintos tipos de selectores más utilizados? Los distintos tipos de selectores más utilizados son: Selector de tipo: Selecciona elementos HTML según su tipo. Ejemplo: p selecciona todos los párrafos. Selector de clase: Selecciona elementos HTML que tienen un atributo de clase específico. Ejemplo: .clase selecciona todos los elementos con la clase "clase". Selector de ID: Selecciona un elemento HTML con un atributo de ID específico. Ejemplo: #id selecciona el elemento con el ID "id".

Ejemplifique cada uno.

5. ¿ Qué es una pseudo-clase? Cuáles son las más utilizadas aplicadas a vínculos? Una pseudo-clase es una palabra clave que se agrega a un selector y especifica un estado especial del elemento seleccionado. Las pseudo-clases más utilizadas aplicadas a vínculos son: :hover: Aplica un estilo cuando el cursor se sitúa sobre el elemento. :active: Aplica un estilo cuando el elemento está siendo activado. :visited: Aplica un estilo a un vínculo visitado.

6. ¿ Qué es la herencia? La herencia en CSS es el mecanismo mediante el cual un elemento hijo hereda ciertos estilos de su elemento padre. Esto significa que si un elemento no tiene un estilo definido para una propiedad particular, heredará ese estilo de su elemento padre.

7. ¿ En qué consiste el proceso denominado cascada? El proceso denominado cascada en CSS se refiere a la manera en que se resuelven los conflictos entre diferentes reglas de estilo que se aplican a un mismo elemento. La cascada sigue un conjunto de reglas específicas para determinar qué estilo prevalece, teniendo en cuenta la especificidad de los selectores, el orden de las reglas y la importancia de las declaraciones. Esto permite que los estilos se apliquen de manera predecible y coherente en un documento CSS.
